Getting started

Intialize the SDK

You can simply initialize the BroseSDK by doing 

let broseSDK = BroseSDK()

When you are having an API Key you can configure the SDK with it:

let apiKey = "YOUR_API_KEY"
let configuration = BroseSDKConfiguration(apiKey: apiKey)
let broseSDK = BroseSDK(configuration: configuration)

If you want to get a German instead of an English localization for the error information you can do this by setting the locale in the configuration:

let configuration = BroseSDKConfiguration(locale: .german)

Attaching a logger

To attach a logger to the SDK you need to implement the BroseLogger protocol:

public protocol BroseLogger {
    func log(message: String, logLevel: BroseLogLevel)
}

After that you can attach the Logger instance to the SDK:

broseSDK.attach(logger: BroseLoggerImpl())

Objective-C support

The SDK is also usable from Objective-C. There are multiple classes specially created to establish the complete functionality of the SDK. They are suffixed with _Objc. For example here is the Objective-C code to initialize the BroseSDK:

#import <BroseSDK/BroseSDK-Swift.h>

BroseSDKConfiguration* config = [[BroseSDKConfiguration alloc] initWithApiKey:@"YOUR_API_KEY" locale:BroseSDKLocaleGerman];
    self.broseSDK = [[BroseSDK_ObjC alloc] initWithConfiguration:config];
