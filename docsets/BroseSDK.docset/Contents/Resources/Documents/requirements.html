Requirements

Minimum iOS Version

Currently the Brose iOS SDK is supporting iOS 12 and later.

Bluetooth

Description in the Info.plist

The app has to provide a text in its Info.plist for the Key NSBluetoothAlwaysUsageDescription to tell the user that we need those rights to communicate with the bike.

Background modes

To work correctly the SDK using app has to set some background modes:

  • External accessory communication
  • Uses Bluetooth LE accessory
  • Background processing
